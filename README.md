# Dev-Agents WSL2

🤖 WSL2 Ubuntu 기반 AI 에이전트 통합 개발 환경

## 🚀 Quick Start

1. WSL2 설치
wsl --install Ubuntu-22.04

2. 프로젝트 클론 및 설정
git clone <repository-url>
cd dev-agents-wsl
bash scripts/setup.sh

3. VS Code로 열기
code .

text

## 📁 구조

- `bin/` - 실행 스크립트
- `scripts/` - 설치 및 관리 스크립트  
- `configs/` - 에이전트별 설정
- `docs/` - 문서
